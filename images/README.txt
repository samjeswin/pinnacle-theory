IMAGES DIRECTORY
================

Please add your logo image here:

1. Logo: Place your company logo as "logo.png" in this directory
   - Recommended: 200x200px or larger
   - Format: PNG with transparent background preferred
   - The website will display the logo in the header

2. Bank Logos (Optional): Add bank logo images to the "banks/" subdirectory
   - Currently using text placeholders
   - Can be replaced with actual bank logos if available

3. Icons (Optional): Add custom icons to the "icons/" subdirectory
   - Currently using emoji icons
   - Can be replaced with custom icon images if preferred

Note: The website is designed to work without images (using text fallbacks),
but adding your logo will enhance the professional appearance.
